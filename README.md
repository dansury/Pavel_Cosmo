# Космоэнергетика · РЕЗОНАНС

Лендинг для Павла Савинкина — целителя-космоэнергета. Концепция выстроена
вокруг базовой идеи космоэнергетики: **резонанса** с космическими каналами
через семь сфер ауры (чакр). Каждый раздел сайта соответствует своей частоте
и цвету — от муладхары к сахасраре.

## Структура

```
.
├── index.html      — разметка лендинга
├── styles.css      — оформление, анимации, чакральная палитра
├── script.js       — звёздное поле, курсор-аура, reveal-анимации
├── images/
│   ├── pavel.jpg   — портрет
│   └── cosmos.jpg  — визуализация
└── .nojekyll       — отключаем Jekyll-обработку для GitHub Pages
```

## Деплой на GitHub Pages

1. Слить ветку `claude/cosmic-landing-page-lljjs` в `main` через PR.
2. В настройках репозитория → **Settings → Pages**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` · `/ (root)` → **Save**.
3. Через 1–2 минуты сайт будет доступен по адресу:
   `https://<owner>.github.io/<repo>/`

Если хочется опубликовать без слияния с `main`, можно прямо в Pages выбрать
текущую ветку — `claude/cosmic-landing-page-lljjs`.
